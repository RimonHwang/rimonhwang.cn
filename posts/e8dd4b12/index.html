    使用Hexo + Github Pages 搭建博客教程

    前言：由于学习Android开发已将近半年，随着知识的不断积累以及代码量的猛增，觉得有必要通过写博客来将这些知识和代码量转变为更加实在的文章，既便于日后查阅，又能练习写作，何乐而不为？

    为什么要自己搭建博客，不用现成平台（CSDN、简书甚至微信公众号）？

    搭建自己的博客平台，是为了享受搭建完成后的一刹快感，同时可定制能够更好地彰显个性。搭建博客并不代表放弃其他平台，我也会把Blog同步至简书（CSDN平台早文章多，但个人觉得界面较老旧），方便互动交流。

    正文

    阅读本文前，你需先掌握 Git和 Github 的相关知识和使用方法，同时为避免重复造轮子，本文仅提供搭建博客的整体思路。当然，通过本文和相关连接，你完全能够搭建出属于自己的博客。

    一、Hexo简介

    Hexo是一个快速、简洁且高效的博客框架，默认使用Markdown解析文章。Hexo官网包括安装及使用教程，提供博客主题和插件下载等，想要个性化使用Hexo可详细查阅Hexo官网相关内容。
